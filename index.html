About the Visualization

Interactive Slideshow This narrative visualization follows an interactive slideshow structure to guide users through various selected scenarios. This is the best choice for this dataset as it is too big to be presented to users directly. Data for each slide has been pre-processed and aggregated separately offline.

Scenes Four scenes have been included in this visualization using consistent template and layout. Each slide can be navigated using arrows at left or right, or using the navigation dots on top. Or use keyboard left and right buttons.

Annotations Interesting trends are annotated using orange color font and box in chart 1, 2 and 3.

Parameters Chart 2 and 3 include filters to display data from selected type of data source. Chart 4 includes a checkbox to reveal / hide grid that covers on top of the map.

Triggers The animation of chart 3 and 4 will be triggered automatically when the scene / slide is displayed. Changing filter in chart 3 will also trigger the chart animation.

Chrome is recommended. Chart 4 may be very slow or unresponsive in other browsers.
